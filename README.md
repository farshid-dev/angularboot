# 🧩 Dynamic Role-Based Web Application

A **Spring Boot** + **AngularJS** + **MySQL** web application that dynamically adjusts its menu structure and feature access based on user roles. Built as a demonstration of **role-based access control (RBAC)** with a fully dynamic UI.

---

## ✨ Features

- 🔐 **Role-Based Access Control (RBAC)** – Menus and features change dynamically based on the user's role.
- 🧭 **Dynamic Menu Rendering** – Admins see 5+ tabs, regular users see only 2 (or whatever you define).
- 🛡️ **Feature-Level Permissions** – Each tab/feature can be toggled on/off per role.
- 🗄️ **Spring Boot Backend** – RESTful APIs with Spring Security.
- 🎨 **AngularJS Frontend** – Single-page application with dynamic UI rendering.
- 🐬 **MySQL Database** – Stores user roles, permissions, and menu configurations.
- 📦 **Maven Build** – Easy to build and run.
